# dins-database
Cataloging system for antiques


Features:
- User-created tags 
- Sort by tag
- Online back up google drive / git hub

Item Entry:
- Item numbers
- Name of item
- Tags
- Type of item
- Approx Date
- Description
- Materials
- Condition report
- Decade
- Related period
- Price bought
- Price sold 
- Date sold
- Database of sold items
- Option for photos
